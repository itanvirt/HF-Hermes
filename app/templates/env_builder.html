Required secrets critical Set in HF Space → Settings → Variables and secrets
HF_TOKEN critical
Hugging Face token with write access. Used to auto-create and upload to the backup dataset.
CLOUDFLARE_WORKERS_TOKEN critical
Cloudflare API token with Edit Workers permission. Auto-deploys a Worker on first boot for keep-awake pings and Telegram API proxy.
TELEGRAM_BOT_TOKEN critical
Bot token from @BotFather. Required to connect the Hermes gateway to Telegram.
TELEGRAM_ALLOWED_USERS critical
Comma-separated Telegram user IDs allowed to send messages to the agent. Get your ID from @userinfobot.
GATEWAY_TOKEN critical
Shared secret protecting the terminal, ENV Builder, and /v1/* LLM relay. Use a long random string.
LLM_MODEL required
Model identifier. Examples: gemini-2.5-flash, gpt-4o, openrouter/google/gemini-2.5-flash, deepseek/deepseek-chat, xai/grok-4.
LLM_API_KEY required
API key for the provider implied by LLM_MODEL. Auto-routed to the correct provider env var on boot.
Telegram feature
TELEGRAM_MODE optional
Delivery method. Leave unset for long-polling (default, recommended). Set to webhook for inbound webhook delivery instead.
TELEGRAM_WEBHOOK_SECRET optional
Secret token Telegram sends in X-Telegram-Bot-Api-Secret-Token. Auto-generated on first boot if not provided.
Startup packages & scripts feature Installed/run on every boot
STARTUP_APT_PACKAGES optional
Space-separated apt packages to install on every boot. Example: ffmpeg jq.
STARTUP_PIP_PACKAGES optional
Space-separated pip packages to install on every boot.
STARTUP_NPM_PACKAGES optional
Space-separated global npm packages to install on every boot.
STARTUP_RUN optional
Bash command(s) to run on every boot. For multi-line scripts use STARTUP_RUN_BASE64.
STARTUP_RUN_BASE64 optional
Base64-encoded multi-line bash script run on every boot. Takes precedence over STARTUP_RUN.
State backup feature
SYNC_INTERVAL optional
Backup frequency in seconds. Default: 600 (10 minutes).
BACKUP_DATASET_NAME optional
Dataset repo name for backups. Owner is auto-detected from HF_TOKEN. Default: hermes-backup.
BACKUP_DATASET_REPO optional
Full dataset repo ID (e.g. myuser/my-backup). Overrides BACKUP_DATASET_NAME.
Provider API keys optional Set per-provider keys, or use comma-separated *_KEYS rotation pools
OPENROUTER_API_KEY optional
API key for OpenRouter (default fallback provider). Also accepts OPENROUTER_API_KEYS comma-separated pool.
GOOGLE_API_KEY optional
API key for Google Gemini. Also accepts GOOGLE_API_KEYS for rotation.
OPENAI_API_KEY optional
API key for OpenAI. Also accepts OPENAI_API_KEYS for rotation.
ANTHROPIC_API_KEY optional
API key for Anthropic. Also accepts ANTHROPIC_API_KEYS for rotation.
DEEPSEEK_API_KEY optional
API key for DeepSeek. Also accepts DEEPSEEK_API_KEYS for rotation.
XAI_API_KEY optional
API key for xAI (Grok). Also accepts XAI_API_KEYS for rotation.
NVIDIA_API_KEY optional
API key for NVIDIA NIM. Also accepts NVIDIA_API_KEYS for rotation.
HF_INFERENCE_TOKEN optional
Hugging Face Inference API token for hf/* model IDs. Can reuse your HF_TOKEN.
Advanced advanced
CLOUDFLARE_KEEPALIVE_ENABLED advanced
Set to false to deploy the Worker (Telegram proxy) without the keep-awake cron trigger. Default: true.
CLOUDFLARE_ACCOUNT_ID advanced
Override the Cloudflare account ID if your token has access to multiple accounts. Auto-detected otherwise.
GATEWAY_RESTART_DELAY advanced
Seconds to wait between gateway restarts on crash. Default: 5.
GATEWAY_MAX_RESTARTS advanced
Maximum restart attempts after crashes. 0 means unlimited. Default: 0.
WEBHOOK_URL advanced
URL to POST to on each gateway restart. Receives a JSON body with event and space fields.
